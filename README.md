# BudgetTracker  

A simple yet powerful application to manage personal finances, providing tools for tracking expenses, managing accounts, and setting budgets effectively.  

## Features  

1. **User Registration & Authentication**  
   - Register with a unique username and password.  
   - Secure login system for personalized access.  

2. **Account Management**  
   - Add, edit, or delete financial accounts (e.g., cash, card, savings).  
   - Set initial balances for better financial oversight.  

3. **Category Management**  
   - Manage expense categories such as food, shopping, bills, etc.  
   - Add, edit, or delete categories as needed.  

4. **Expense Management**  
   - Log expenses by specifying date, amount, category, and description.  
   - View all recorded expenses by category or date range.  

5. **Budget Management**  
   - Set budgets for different categories.  
   - View and update budgets to stay on top of spending goals.  

## Technology Stack  

- **Front-End**: Java Swing  
- **Back-End**: MySQL 8.0.39  

## How to Use  

1. Clone the repository.  
2. Set up the MySQL database using the provided schema.  
3. Run the application using any Java IDE or terminal.  
4. Register, log in, and start managing your finances!
